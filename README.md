# Benefon Bot — Telegram бот для управления строительными проектами

**Версия:** 1.0.0  
**Дата:** 18.06.2026  
**Автор:** Агент-архитектор

## 📋 Описание

Telegram-бот для управления строительными проектами с веб-панелью. Система включает:

- ✅ Управление задачами (создание, назначение, фотоотчёты, утверждение, оплата)
- ✅ Учёт материалов (остатки, критические уровни, заявки)
- ✅ Управление инструментами (назначение, возврат, история)
- ✅ 6 ролей пользователей (владелец, гендиректор, ПТО, прораб, электрик, рабочий)
- ✅ Веб-панель для просмотра отчётов и статистики
- ✅ Автоматические отчёты (PDF, Excel, графики)
- ✅ Уведомления и дайджесты

## 🚀 Быстрый старт

### 1. Клонирование репозитория

```bash
git clone https://github.com/SamBrawn/benefon-bot.git
cd benefon-bot
```

### 2. Установка зависимостей

```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 3. Настройка окружения

```bash
cp .env.example .env
```

Отредактируйте `.env`:
```env
BOT_TOKEN=ваш_токен_бота
DATABASE_URL=postgresql://...
ADMIN_IDS=ваш_telegram_id
```

### 4. Инициализация базы данных

```bash
# Создание таблиц (для разработки)
python -c "from database import init_db; import asyncio; asyncio.run(init_db())"
```

### 5. Запуск

```bash
# Локальный режим (polling)
python main.py
```

Бот запустится на `http://localhost:8000`

## 🛠️ Технологии

- **Python 3.11+**
- **aiogram 3.x** — Telegram бот
- **FastAPI** — веб-панель и API
- **SQLAlchemy 2.0** — ORM
- **PostgreSQL (Neon.tech)** — база данных
- **Alembic** — миграции
- **APScheduler** — планировщик задач
- **FPDF2** — генерация PDF
- **OpenPyXL** — генерация Excel
- **Matplotlib** — графики
- **Pillow** — обработка фото
- **Loguru** — логирование
- **Docker** — контейнеризация
